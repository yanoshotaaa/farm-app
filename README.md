# 農業作物管理アプリ

本格的な農業作物管理アプリケーションです。作物の登録、成長記録、タスク管理、統計表示などの機能を提供します。

## 機能

### 主要機能

- **作物管理**: 作物の登録、編集、削除
  - 画像アップロード機能（Base64形式で保存）
  - 品種、場所、植え付け日、収穫予定日の管理
  - 状態管理（成長中、収穫済み、除去済み）
- **成長記録**: 作物の成長過程を記録（高さ、幅、メモ、画像）
- **タスク管理**: 水やり、施肥、剪定などのタスクを管理
  - 期限管理と完了機能
  - 緊急タスクの表示
- **カレンダービュー**: 植え付け日、収穫日、タスクをカレンダーで可視化
- **畑・区画管理**: 畑や区画の情報を管理
- **ダッシュボード**: 統計情報とグラフ表示
  - 状態別の円グラフ
  - 月別の植え付け数グラフ
  - 近々の収穫予定（7日以内）
  - 緊急タスク一覧
- **データ管理**: データのエクスポート/インポート
  - JSON形式でのエクスポート/インポート
  - CSV形式でのエクスポート（作物データ）
- **チャット機能**: 作物について質問できるチャットボット
- **検索・フィルター**: 作物の検索と状態別フィルター
- **ユーザー認証**: Firebase Authenticationによるログイン/サインアップ

### 技術スタック

- **React 18** - UI フレームワーク
- **TypeScript** - 型安全性
- **Vite** - ビルドツール
- **Tailwind CSS** - スタイリング
- **Zustand** - 状態管理
- **Firebase** - Firestore（データベース）とAuthentication（認証）
- **React Router** - ルーティング
- **Recharts** - グラフ表示
- **date-fns** - 日付処理
- **Lucide React** - アイコン

## セットアップ

### 必要な環境

- Node.js 18以上
- npm または yarn

### インストール

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## 使い方

### 作物の登録

1. 「作物一覧」ページから「新しい作物を追加」をクリック
2. 作物名、品種、場所、植え付け日、収穫予定日を入力
3. 必要に応じてメモを追加
4. 「追加」ボタンをクリック

### 成長記録の追加

1. 作物の詳細ページを開く
2. 「成長記録」セクションの「記録を追加」をクリック
3. 日付、高さ、幅、メモを入力
4. 「追加」ボタンをクリック

### タスクの管理

1. 「タスク」ページまたは作物の詳細ページからタスクを追加
2. タスクの種類（水やり、施肥、剪定など）を選択
3. 期限を設定
4. タスク完了時に「完了」ボタンをクリック

### カレンダービュー

- 月単位で植え付け日、収穫日、タスクを表示
- イベントを色分けして表示（植え付け: 緑、収穫: 青、タスク: 黄）
- 月の移動と今日へのジャンプ機能

### データ管理

- **エクスポート**: すべてのデータをJSON形式、または作物データをCSV形式でエクスポート
- **インポート**: JSON形式のバックアップファイルからデータを復元
- **データ削除**: すべてのデータを削除（注意: 取り消せません）

### チャット機能

- 「チャット」ページから作物について質問できます
- 作物名や品種を言うと、その作物の情報を表示します
- メッセージ履歴は自動保存されます

## データ保存

データは**Firebase Firestore**に保存されます。以下のメリットがあります：

- ✅ 複数デバイス間でデータ同期
- ✅ データの自動バックアップ
- ✅ ユーザー認証によるデータ保護
- ✅ 容量制限なし（無料枠内）

画像データもBase64形式でFirestoreに保存されます（最大5MB、自動圧縮機能付き）。

### Firebase設定

Firebaseを使用するには、設定が必要です。詳細は [FIREBASE_SETUP.md](./FIREBASE_SETUP.md) を参照してください。

**クイックセットアップ**:
1. [Firebase Console](https://console.firebase.google.com/)でプロジェクトを作成
2. Firestore Databaseを作成
3. Authentication（メール/パスワード）を有効化
4. `.env` ファイルを作成し、Firebase設定を追加
